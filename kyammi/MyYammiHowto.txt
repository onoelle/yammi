yammi - Howto
-------------

i18n
----
call manually in project dir:
make -f admin/Makefile.common package-messages
=> generates yammi.pot in the po directory

old: (QT only)

nach autoconf/automake:
im Verzeichnis yammi/yammi (mit den ganzen sourcen) Makefile.am editieren:
TRANSLATIONS = yammi_de.ts yammi.hu.ts yammi_xx.ts


im Verzeichnis yammi/yammi (mit den ganzen sourcen): lupdate Makefile.am
=> erzeugt/aktualisiert "yammi_de.ts" (+ yammi_xx.ts)
(weil im Makefile.am eingetragen ist: TRANSLATIONS = yammi_de.ts)

im gleichen Verzeichnis lrelease Makefile.am
=> erzeugt yammi_de.qm (+ yammi_xx.qm)


CVS-Howto
---------
dies kreiert lediglich den account auf cvs.sourceforge
ssh -l onoelle cvs.yammi.sourceforge.net

import: bob -> cvs@sourceforge
cvs -d:ext:onoelle@cvs.yammi.sourceforge.net:/cvsroot/yammi import mycvsroot vendor start

checkout: cvs@sourceforge -> bob
cvs -d:ext:onoelle@cvs.yammi.sourceforge.net:/cvsroot/yammi co mycvsroot

checkin (=commit): bob -> cvs@sourceforge
cvs commit -m "comments for this commit" uic_command.txt


Compile Farm benutzen
---------------------
auf compile farm einloggen:
ssh -lonoelle cf.sourceforge.net

dann erstmal:
cvs -d:pserver:anonymous@cvs.yammi.sourceforge.net:/cvsroot/yammi login
(und bei Frage nach Passwort einfach Return)

und schliesslich yammi vom cvs getten:
cvs -z3 -d:pserver:anonymous@cvs.yammi.sourceforge.net:/cvsroot/yammi co HEAD
