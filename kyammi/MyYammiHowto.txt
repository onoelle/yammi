yammi - Howto
-------------

CVS-Howto
---------
dies kreiert lediglich den account auf cvs.sourceforge
ssh -l onoelle cvs.yammi.sourceforge.net

import: bob -> cvs@sourceforge
cvs -d:ext:onoelle@cvs.sourceforge.net:/cvsroot/yammi import kyammi vendor start

checkout: cvs@sourceforge -> bob
cvs -d:ext:onoelle@cvs.sourceforge.net:/cvsroot/yammi co kyammi

checkin (=commit): bob -> cvs@sourceforge
cvs commit -m "comments for this commit" uic_command.txt


Compile Farm benutzen
---------------------
auf compile farm einloggen:
ssh -lonoelle cf.sourceforge.net

dann erstmal:
cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/yammi login
(und bei Frage nach Passwort einfach Return)

und schliesslich yammi vom cvs getten:
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/yammi co kyammi
