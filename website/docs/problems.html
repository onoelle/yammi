Problems

If you experience problems with compiling or installing, please consult the README file included in the distribution which explains various switches for the configure script that might be necessary on your system. If you still experience problems, please have a look at the "Bugs" page on the Yammi homepage for known problems or follow the procedure described on that page for submitting a bug report.
