Using yammi 1.2.1

Searching

Type in anything into the search field, Yammi immediately shows you the best matches, rated with a percentage (you can configure the threshold for the shown matches in the settings). That's basically it, simple but effective, just give it a try! Yammi looks into title, artist, album and comment for similarities. Yammi will automatically switch to the folder "Search Results" whenever you type in anything into the search field. Tip: For quickly searching for a song, use Ctrl-F or ESC to set the focus to the search field and clear it.
