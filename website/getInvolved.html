Get involved in Yammi development...

Are you interested in making Yammi better?

Even if you are not a programmer, there are a lot of things where you could help the project:

  • web design: if you would like to improve the homepage of yammi (or create a completely new one): go for it!
  • graphics: new icons for the application, graphics for the website, ...
  • translations: the KDE translation scheme makes it easy to translate Yammi into your favourite language, see the Translation page for details
  • documentation: the handbook could need some update and polishing...
  • coding: see the TODO file (current version in CVS), there are more than enough features to add and bugs to fix...
  • creating rpms for various distributions
Please contact the yammi-developer mailing list if you would like to contribute, any help is appreciated!
