RPMs

As I don't know anything about packaging Yammi into a rpm, I don't provide any rpm packages myself. However, I am more than happy to make rpm packages of Yammi packed by others available here:
  • yammi-1.2-0.rc2.2mdk.i586.rpm (thanks to thelinuxfr.org), supposed to work on Mandrake 10.2
  • yammi-1.2beta-1.deb (thanks to Jan Suhr), supposed to work on Debian testing/sarge
  • SUSE rpms can be found on http://packman.links2linux.de
other locations (possibly not having the latest version, but worth having a look at):
  • http://packman.links2linux.de has a link to SuSE binary and source rpms of Yammi.
  • You can also find rpm packages of Yammi on rpmseek.com.

CVS

For the newest (and possibly unstable) version in CVS, use the "kyammi" module in Sourceforge's CVS repository.
It's easy:
Simply execute the following command to retrieve the current CVS version:
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/yammi co kyammi
Now you should have a "kyammi" directory under the current directory containing the newest yammi sources, and you can go ahead with the usual configure and make procedure (see the installation chapter of the yammi handbook).
