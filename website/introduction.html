mission
Make music management on your computer as easy and efficient as it should be.

metric
How do we define easy and efficient?

What about measuring the time and ease of use for the following use case:

  1. search for a song "It's Build" by "Hausmartins"
    (or similar, because you can't remember the exact spelling...)
  2. find out that there is a song "Build" by "The Housemartins"
    (oh right, that was the correct name...)
  3. play this song after the currently playing song has finished
    (you don't want to interrupt the lovely song you're listening to right now...)
  4. find all songs on the same album as this song
    (because you want to listen to all of them now...)
  5. and finally burn an audio CD with that album
    (because your girl friend likes that album, but she hates computers...)

solution
Let's see how we do this with Yammi:

Step 1: search for the song
Press "ESC" from anywhere in Yammi to clear the search field and set the focus to it.
Type "It's build Hausmartins"
=> 3-5 seconds (depending on how fast you type)

Step 2: find the song
Look at the best matches list that Yammi automatically provides
=> 1-2 seconds (depending on how fast you look, the search takes much less than a second)

Step 3: play as next song
Choose "Play/Enqueue" -> "Enqueue as next" from the song context menu or the corresponding keyboard shortcut.
=> 1-3 seconds

Step 4: find all songs on the same album
Choose "Go to folder" -> "album" from the song context menu
=> 3 seconds

Step 5: burn an audio CD with that album Choose "Content" -> "Plugins" -> "Burn to audio CD" from the folder context menu.
Start burning process in K3B (which starts up with the desired songs added automatically).
=> 7 seconds
