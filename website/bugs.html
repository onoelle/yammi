Bugs

For viewing open bugs and reporting new bugs, please use the tracker system of sourceforge or report them via email to the yammi-developer mailing list.

When reporting bugs, please include the following information:

  • yammi version (or date of CVS checkout)
  • QT and KDE version
  • the linux distribution you are using
  • for compile problems: the output of the configure and make command
  • for other problems: the console output of Yammi (ie. start yammi in a shell) when the bug occurred
Please try to be as precise as possible in describing what you did and what the error was, so we can quickly address this problem.
